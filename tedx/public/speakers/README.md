- `speaker1.jpg` - Dr. Sarah Mitchell (Large card - 2x2)
- `speaker2.jpg` - James Chen (Medium card - 1x1)
- `speaker3.jpg` - Maria Rodriguez (Tall card - 1x2)

- `speaker5.jpg` - Emma Thompson (Medium card - 1x1)
- `speaker6.jpg` - Dr. Yuki Tanaka (Medium card - 1x1)
- `speaker7.jpg` - Marcus Williams (Large card - 2x2)
- `speaker8.jpg` - Sofia Petrov (Medium card - 1x1)

## Image Requirements:
- Format: JPG, PNG, or WebP
- Minimum dimensions: 800x600px
- Aspect ratio: Varies based on card size (larger cards need higher resolution)
- Professional, high-quality photos
- Portrait or landscape orientation both work

## Tips:
- Use consistent image quality across all speakers
- Ensure good contrast for text overlay visibility
- Consider using images with darker backgrounds or subjects positioned to leave space for text
