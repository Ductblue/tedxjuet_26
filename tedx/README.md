# TEDxMiami Premium Website 🔴⚫

A premium, fully-animated recreation of the TEDxMiami website built with Next.js, TypeScript, and Framer Motion.

## ✨ Premium Features

### 🎨 Advanced Animations
- **Parallax Hero Section** - Smooth scrolling effects with floating circles
- **Staggered Fade-ins** - Sequential animations for all content sections
- **Hover Effects** - Scale, lift, and glow animations on interactive elements
- **Scroll-triggered Animations** - Elements animate as they enter viewport
- **Smooth Transitions** - Fluid animations using Framer Motion

### 🎯 Design Features
- **Modern Red & Black Theme** - Premium color palette matching TEDx branding
- **Glassmorphism Effects** - Frosted glass navbar with blur effects
- **Gradient Overlays** - Dynamic gradients on hero and event sections
- **Responsive Design** - Perfectly adapts to all screen sizes
- **Custom Typography** - Inter font for modern, clean look

### 🚀 Technical Features
- **Next.js 14** - Latest React framework with App Router
- **TypeScript** - Type-safe development
- **Framer Motion** - Professional animation library
- **GSAP** - Advanced animation capabilities
- **CSS Modules** - Scoped styling for components
- **Mobile Navigation** - Smooth hamburger menu with animations

## 🛠️ Installation & Setup

```bash
# Navigate to the project
cd tedxmiami-premium

# Install dependencies (already done)
npm install

# Run development server
npm run dev
```

Visit: **http://localhost:3001**
